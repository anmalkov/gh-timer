# gh-timer

A gh extension that runs timer or stopwatch

## installation

gh extension install anmalkov/gh-timer

## usage

- `gh timer --sec 10` - run a stopwatch for 10 seconds
- `gh timer --min 5` - run a stopwatch for 5 minutes
- `gh timer --help` - show help
